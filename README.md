### 🛠 Installation

```sh
git clone https://github.com/spicesouls/exodus
cd exodus && pip install -r requirements.txt
chmod +x exodus.py
```

### 📃 Usage

```sh
./exodus.py [ IP / CIDR ] [ PROTOCOL ] -u [ USERNAME ] -p [ PASSWORD ] -t [ THREADS ]
```
